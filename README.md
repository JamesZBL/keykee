# keykee
这是一个记录键盘的使用情况的软件

## 安装

1.克隆代码仓库到本地

```sh
$ git clone https://gitee.com/zbl1996/keyee.git
$ cd keykee
```

2.安装 pipreqs
```sh
$ pip install pipreqs
```

3.安装项目依赖

```sh
$ pip install -r requirements.txt
```

4.编译安装
```sh
$ sudo ./install
```
